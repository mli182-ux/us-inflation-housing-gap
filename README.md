# US Inflation and Housing Cost Comparison

This project compares U.S. inflation, housing costs, and income growth.

## Research Question
Have U.S. housing costs grown faster than overall inflation and income?

## Data Sources
- FRED CPI data
- Zillow Home Value Index
- Census / FRED income data

## Methods
- Data cleaning
- Trend visualization
- Housing-to-income comparison
- Regression analysis

## Goal
Create visualizations and a one-page project report explaining why housing costs have grown faster and how this affects students.
